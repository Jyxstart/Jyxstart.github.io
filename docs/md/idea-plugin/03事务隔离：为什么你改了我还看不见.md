## 03 | 事物隔离:为什么你改了我还看不见？

事物：A(原子性)C(一致性)I(隔离性)D(持久性)

|          | 脏读  | 不可重复读 | 幻读  |
| :------: | :---: | :--------: | :---: |
| 读未提交 | **×** |   **×**    | **×** |
|  读提交  | **√** |   **×**    | **×** |
| 可重复读 | **√** |   **√**    | **×** |
|  串行化  | **√** |   **√**    | **√** |

脏读:一个事物读取到了另一个事物已经修改但未提交的数据。

不可重复读:一个事物第一次读到的数据与之后读取的数据不一致。(数据的修改)

幻读:一个事物读取到了另一个事物新增的数据。(数据的新增)
